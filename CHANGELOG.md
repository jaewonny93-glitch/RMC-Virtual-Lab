# RMC Virtual Lab - 변경 이력

## v1.0.0 (2025-07-01) - 최초 릴리즈

### 🎉 새 기능
- 세포 배양 전체 워크플로우 구현 (클린벤치 → 인큐베이터)
- 55종 세포주 데이터베이스 (줄기세포 / 암세포주 / 정상세포주 / 동물세포주 / 면역 / 신경 / 간 / 심장/근육 / 골/연골)
- 실시간 합류도(Confluence) 계산 (더블링타임 × dish 면적 기반 로지스틱 성장곡선)
- 배양액 교체 알림 (72h 기준, 144h 초과 시 세포 사멸 처리)
- 계대배양(Sub-culture) 전체 워크플로우 (원심분리 → 세포 수 계측 → 재파종)
- 헤모사이토미터 트리판 블루 세포 수 계산기
- 실험 노트 자동 생성 및 PNG 이미지 내보내기
- 진행 중 배양 홈화면 실시간 모니터링
- 성장 곡선 그래프 (실시간 + 예측선)
- 실험 기록 히스토리 관리
- 사용자 승인 시스템 (연구원 / 선임연구원 / 교수 / 기타)
- 관리자 모드 (사용자 승인 관리, 공지사항)
- iOS / Android 홈화면 추가 QR 코드

### 🏗️ 기술 스택
- Flutter 3.35.4 / Dart 3.9.2
- Firebase Firestore (사용자 인증 및 데이터 동기화)
- Provider 상태 관리
- fl_chart 성장 곡선 시각화
